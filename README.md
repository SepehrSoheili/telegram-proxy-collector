# telegram-proxy-collector
Collect telegram channel proxy with simple php class

#### types :
- Mtproto
- Socks 
- and any proxy in channel

#### proxy in :
- message text
- inline button 
- url hrefed 
- and any proxy in channel

## how to use :
add `telegramProxyCollector.php` to your code like `example.php` and enjoy it !
